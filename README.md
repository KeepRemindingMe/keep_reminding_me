### How to run locally

Create a virtual environment 

```
>>> virtualenv env
>>> source env/bin/activate
```

Install the dependencies

```
>>> pip install -r requirements.txt
```

Ensure your keys are in your secrets.sh file and run them.
```
>>> source secrets.sh
```
